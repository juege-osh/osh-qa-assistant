# AI Assistant 本地统一启动说明

## 当前结构

- 后端统一入口：`backend`
- 管理端前端：`frontend/ai-assistant-manager`
- 用户端前端：`frontend/ai-assistant-consumer`

`manager`、`consumer`、`common` 代码已经统一由 `backend` 这个单一 Spring Boot 进程承载。

## 访问地址

- 管理端前端：`http://localhost:8001/`
- 用户端前端：`http://localhost:9001/`
- 统一后端健康检查：`http://localhost:9000/`
- 管理端接口前缀：`http://localhost:9000/manager`
- 用户端接口前缀：`http://localhost:9000/consumer`

## 启动前要求

1. JDK 17
2. Node.js / npm
3. Redis 可连通
4. MySQL 可连通
5. Qdrant 可连通

当前默认配置在：

- [`backend/src/main/resources/application.yml`](/Users/rengang/chuangye/osh-projects/ai-assistant/backend/src/main/resources/application.yml)

## 构建后端

在项目根目录执行：

```bash
export JAVA_HOME=/Users/rengang/Library/Java/JavaVirtualMachines/ms-17.0.17/Contents/Home
export PATH=/Users/rengang/chuangye/osh-projects/.tools/apache-maven-3.9.9/bin:$JAVA_HOME/bin:$PATH
mvn -f pom.xml -pl backend -am package -DskipTests
```

## 启动方式

推荐直接分别启动 3 个进程。

### 1. 启动统一后端

```bash
export JAVA_HOME=/Users/rengang/Library/Java/JavaVirtualMachines/ms-17.0.17/Contents/Home
$JAVA_HOME/bin/java -jar backend/target/backend.jar
```

### 2. 启动管理端前端

```bash
cd frontend/ai-assistant-manager
npm run dev -- --host 0.0.0.0
```

### 3. 启动用户端前端

```bash
cd frontend/ai-assistant-consumer
npm run dev -- --host 0.0.0.0
```
